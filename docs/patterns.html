The Minesweeper Patterns

Minesweeper is filled with various patterns and recognizing them can help you become a great player! Here are a few:

B1

If a number is touching the same number of cells, then all of those cells are mines.

B2

If a number is touching the same number of flagged mines, then any other uncovered square around it is safe.

1-1+

1-2-1

If you come across a pattern of alternating 1s and 2s that starts and ends with 1, it is assured that there are mines directly beside the 1s and the tiles beside the 2s are safe.

1-2-2-1

In a 1-2-2-1 pattern, there is only one solution. Since there needs to be two mines beside each of the 2s but only one mine beside to the 1s, the only place for the mines to be would be directly beside the 2s

H1 and H2

H1 and H2 are patterns that lead from one to the other. H1: If you are able to clear a tile in a wall that reveals a 1 and you are sure that one of the tiles beside it have a bomb, you can assume that the tiles behind the 1 are safe. The middle tile that is then revealed is a 1 the same process can determine that the 3 tiles behind it are safe.
